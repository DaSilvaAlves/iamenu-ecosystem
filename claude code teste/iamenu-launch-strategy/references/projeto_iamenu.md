# Informação do Projeto iamenu

## Visão Geral do Projeto

O iamenu é um sistema de restaurante futurista que utiliza avatares multimodais e inteligência artificial para transformar a experiência de atendimento em restaurantes.

## Problema que Resolve

### Desafios Restaurantes Tradicionais

1. **Falta de personalização**
   - Menus estáticos e genéricos para todos os clientes
   - Longas filas / esperas

2. **Impactos na Eficiência**
   - Perda de Clientes Repetidos
   - Desperdício de Recursos (funcionários mal treinados)
   - Tarefas repetitivas financeiras
   - Dificuldades maximizar lucro (pratos estratégicos)

3. **Atendimento**
   - Falta de Conexão Emocional
   - Descontinuidade no Serviço (rotatividade funcionários)
   - Dificuldade Gerir Picos (época alta)
   - Impacto na Reputação (avaliações online)

4. **Como Funciona**
   - Menu digital interativo via QR Code
   - Avatar multimodal/humanizado para assistência personalizada
   - Sugestões estratégicas (aumentar margem)
   - Comunicação fluida (clientes, sala, cozinha)
   - Pedidos enviados diretos para a cozinha
   - Atendimento constante e escalável

## Solução: Chefe de Sala Virtual

### Adopção

**Para o Cliente:**
- Melhorar experiência (ágil, personalizada)
- Sugestões baseadas em preferências
- Menor Envolvimento do Cliente (sugestões/explicações)
- Maior Enrolamento do Cliente

**Para o Restaurante:**
- Redução de custos (menos funcionários, sem comissões)

- Perda de Identidade Local

## Tecnologia Utilizada

- **IA Multimodal**: Processamento de voz, texto, e imagens
- **PNL (Processamento Linguagem Natural)**: Compreensão de pedidos complexos
- **Integração sistemas internos**: ERP, POS
- **Venda estratégia diferenciada hospitalidade**
- **Venda pelo carro**
- **Encaixa cliente histórico, ingredientes**
- **Funcionalidades alivadas**

## Funcionalidades Principais

1. **Menu QR Codes**: Menu digital via avatar (cozi, digital)
2. **Entrada: Interação com avatar (cozi, digital)**
3. **Carrinho: Pedidos recebidos em tempo real**
4. **Sem comissões por pedido**
