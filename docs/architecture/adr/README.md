# Architecture Decision Records (ADRs)

Este diretório contém os registos de decisões arquiteturais do iaMenu Ecosystem.

## O que é um ADR?

Um ADR (Architecture Decision Record) documenta uma decisão arquitetural significativa, incluindo o contexto, a decisão tomada, e as consequências.

## Status dos ADRs

- **Proposto**: Decisão em discussão
- **Aceite**: Decisão aprovada e implementada
- **Deprecado**: Decisão substituída por outra
- **Rejeitado**: Decisão considerada mas não adotada
